Funcionalities
    CD:
        - change directory
        - absolute, relative, /, -, ..
    MKDIR:
        - Make Directory
        - multiple at one
        - prompt before replacing
    RM:
        - Remove Element
        - multiple at once
    MV:
        - move element
        - prompt before replacing
    PWD:
        - print working directory
        - writes to stdout
    LS: 
        - list elements
        - -l long format
        - writes to stdout
    CP:
        - copy element
    GREP:
        - filter 
        - -r search recursively
        - -i search caseInsensitive
        - -v search not non-matches
        - writes to stdout
        - reads from stdout
    ECHO:
        - print entered string
        - writes to stdout
    EXECUTE:
        - execute file
    MAN:
        - get manual
    TOUCH: 
        - create file
        - open file
        - edit and save file
    CAT: 
        - get file content
        - writes to stdout
    FIND: 
        - get all paths
        - filter by name 
        - search name wildcard
        - writes to stdout
    HEAD: 
        - get n number of lines from start file
    TAIL: 
        - get n number of lines from end of file
    |:
        - pipe commands
    &&: 
        - run multiple commands
    >>:
        - redirect output to append(>>)/overwrite(>) scrolls
    ||:
        - failsafe
